# mds_ucboulder_nypd_analysis
Final project for UC Boulder's MS-DS course "Data Science as a Field".

## Motivation
For my final project, I utilized the historic NYPD shooting data to better understand whether race-on-race crime by the same race
was more prevelant than by different races. Furthermore I wanted to better understand if the type of race-on-race crime varied
depending on the borough. The rationale of this question was whether we can gain insights into underlying socioeconomic challenges
certain ethnicities may be facing that could be addressed more specifically in thier respective boroughs.
